Access.Database (Power Query)

Connect to Microsoft Access (.mdb / .accdb) databases in Power Query.

TL;DR:

Access.Database(File.Contents("C:\Data\file.accdb"))

This returns a navigation table where you can select tables or queries inside the Access database.

Syntax

Access.Database(file as binary, optional options as record) as table

Parameters

Parameter Type Required Description
file binary Yes Use File.Contents("path")
options record No Optional settings like navigation properties

Basic Example

let
    Source = Access.Database(File.Contents("C:\Data\example.accdb")),
    Customers = Source{[Schema="", Item="Customers"]}[Data]
in
    Customers

What happens:

  • Connects to Access file
  • Lists tables
  • Extracts "Customers" table

Example — With Navigation Properties

let
    Source = Access.Database(
        File.Contents("C:\Data\example.accdb"),
        [CreateNavigationProperties=true]
    ),
    Orders = Source{[Item="Orders"]}[Data]
in
    Orders

Use this when working with relationships between tables.

Example — Dynamic File Path

let
    Path = Excel.CurrentWorkbook(){[Name="DBPath"]}[Content]{0}[Path],
    Source = Access.Database(File.Contents(Path & "\example.accdb")),
    Data = Source{[Item="Sales"]}[Data]
in
    Data

This allows users to change the file location without editing code.

Example — Combine Multiple Access Files

let
    Folder = Folder.Files("C:\Data\AccessFiles"),
    Filtered = Table.SelectRows(Folder, each Text.EndsWith([Name], ".accdb")),

    AddTables = Table.AddColumn(
        Filtered,
        "Tables",
        each Access.Database(File.Contents([Folder Path] & [Name]))
    ),

    Expanded = Table.ExpandTableColumn(
        AddTables,
        "Tables",
        {"Item", "Data"}
    ),

    CustomersOnly = Table.SelectRows(Expanded, each [Item] = "Customers"),

    Combined = Table.Combine(CustomersOnly[Data])
in
    Combined

This is one of the most powerful real-world use cases.

Common Errors & Fixes

1. Invalid Path

Error: path must be absolute

Fix: Always use full path inside File.Contents()

2. Permissions / Privacy

Power Query may block access. Adjust Privacy Levels or enable fast combine.

3. Power BI Service Refresh Fails

Use an On-Premises Data Gateway for scheduled refresh.

Performance Tips

  • Use Access queries to pre-filter data
  • Avoid loading full tables if not needed
  • Combine files only after filtering

When to Use Access.Database

  • You have .accdb / .mdb files
  • You want structured navigation (tables, queries)
  • You want no SQL required

Alternatives

  • Odbc.Query → more control
  • OleDb → legacy scenarios
